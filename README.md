音频介绍
--------
Android平台中音频播放主要有以下两种方式：

1- SoundPool声音池：适合短促且对反应速度比较高的情况（游戏音效或按键声等）

2- MediaPlayer媒体播放器：适合比较长且对反应速度要求不高的的情况（音乐等）
